# BaksDev Api Wildberries 

[![Version](https://img.shields.io/badge/version-7.1.6-blue)](https://github.com/baks-dev/wildberries/releases)
![php 8.3+](https://img.shields.io/badge/php-min%208.3-red.svg)

Модуль Wildberries Api

## Установка

``` bash
$ composer require baks-dev/wildberries
```

## Дополнительно

Установка конфигурации и файловых ресурсов:

``` bash
$ php bin/console baks:assets:install
```

## Тестирование

``` bash
$ php bin/phpunit --group=wildberries
```
